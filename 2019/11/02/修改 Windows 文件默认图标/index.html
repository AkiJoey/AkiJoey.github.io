修改 Windows 文件默认图标
Created 2019-11-02| Updated 2020-02-20|Common
Word count: 655|Reading time: 3 min|Post View:

修改 Windows 文件的默认图标.
可以分为以下两个步骤:

  • 批量图片格式转换
  • 批量修改注册表

批量图片格式转换

这里使用的是 vscode-icons 插件中的部分图标.
因为这里的图标比较全, 当然你也可以用自己喜欢的图标.
插件中的图标都是 svg 格式, 所以要将其转换为 ico 格式.

安装 Wand

这里要用到 python 的一个图片处理模块 Wand.

$ pip install Wand

安装 ImageMagick

Wand 需要与 ImageMagick 连接使用, 因此还需要安装 ImageMagick.
具体安装过程见: http://docs.wand-py.org/en/latest/guide/install.html

convert 命令可以直接转换图片格式.

$ convert -background transparent a.svg a.ico

批量修改注册表

操作注册表需要用到 python 的内置模块 winreg.
官方文档: https://docs.python.org/3.7/library/winreg.html

单次更新

思路就是修改 HKEY_CLASSES_ROOT 下的某些默认值
(同时实现将文件的默认打开方式设置为 VSCode)
一个文件类型具体要实现三次注册表更新, 以 .c 为例

  • 修改 .c 的默认值为 .c_auto_file
  • 修改 .c_auto_file\Defaulticon 的默认值为 图标路径
  • 修改 .c_auto_file\shell\open\command 的默认值为 VSCode 路径
