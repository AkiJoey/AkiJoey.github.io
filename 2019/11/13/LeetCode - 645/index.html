Created 2019-11-13| Updated 2020-02-18|Algorithm

Problem

题目链接: https://leetcode.com/problems/set-mismatch/

Write Up

这道题像是 LeetCode - 442 和 LeetCode - 448 的结合体.
丢失了一个元素并且有一个元素重复.
可以利用正负性来判断元素是否存在.
若 nums[abs(nums[i]) - 1] < 0.
则该元素重复出现, 即为重复的元素.
若 nums[i] > 0.
则该元素未出现, 即为丢失的元素.
