本文为该项目的具体开发文档.

前后端项目整合

Rails 项目已经整合了部分前端资源.
只需要安装前端依赖即可.

安装 Rails 依赖

$ gem install rails

新建 Rails 项目

新建 Rails 项目, 数据库使用 mysql.

$ rails new TimeControl -d mysql

安装 gem 依赖

$ bundle install

安装 node 依赖

$ npm install

安装 node 项目依赖于 node_modules 目录.

安装 angular 依赖

$ rails webpacker:install:angular
